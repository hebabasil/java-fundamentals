# java-fundamentals

## 📌 Overview

This repository contains Java fundamental programming tasks that help build logical thinking and programming skills before starting Automation Testing in SQA.

These tasks focus on core concepts such as conditions, loops, arrays, exception handling, and basic problem solving.

---

## 📚 Tasks Included

### 1. If-Else Statements

Practicing decision making using conditional statements.

---

### 2. While Loop

Repetition of code execution based on a condition.

---

### 3. FizzBuzz

A logic-based problem:

* Print "Fizz" for multiples of 3
* Print "Buzz" for multiples of 5
* Print "FizzBuzz" for multiples of both
* Otherwise print the number

---

### 4. For-each Loop with 2D Array

Iterating through a two-dimensional array using enhanced for-loop.

---

### 5. Array Read & Write

Accessing and modifying array elements.

---

### 6. Equals Method

Using `.equals()` for proper object comparison.

---

### 7. HashMap Example

Storing and retrieving key-value pairs using HashMap.

---

### 8. Exception Handling (Negative & Index Errors)

Handling:

* Negative values
* Array index out of bounds

---

### 9. Swapping Numbers (Using 3 Variables)

Swapping two numbers using a temporary variable.

---

### 10. Swapping Numbers (Without Variable)

Swapping two numbers without using a third variable.

---

### 11. Order & Item Challenge

Basic logic for managing items and orders.

---

### 12. Barista Challenge

Simulating order processing in a simple system.

---

### 13. Even or Odd Checker

Determining whether a number is even or odd.

---

### 14. Factorial Calculation

Calculating factorial using loops or recursion.

---

## 🎯 Purpose

These tasks are part of Java fundamentals training to prepare for Software Quality Assurance (SQA) Automation Testing.

They help in building:

* Logical thinking
* Problem solving
* Understanding of programming basics
